* Yadm dotfiles

Personal dotfiles managed by [[https://yadm.io][yadm]].

** Setup

Run ~yadm clone~ and then ~yadm bootstrap~. The bootstrap script installs
[[https://mise.run][mise]] and core tools, then hands off to an LLM agent with
[[file:~/.config/yadm/bootstrap-prompt.md][bootstrap-prompt.md]] for interactive
setup.

See [[file:~/.config/yadm/bootstrap][~/.config/yadm/bootstrap]] for the entry point.

** Reference

- [[file:~/.config/yadm/notes.org][~/.config/yadm/notes.org]] — yadm classes, transcrypt, alt files, zsh secrets, Kinesis Advantage, Mac setup notes
- [[file:~/.config/yadm/remote-setup.md][~/.config/yadm/remote-setup.md]] — provisioning a new remote dev machine (Hetzner, tailscale, user setup, bootstrap)
- [[file:~/linux_laptop.org][~/linux_laptop.org]] — Linux laptop config (suspend/hibernate, power management, audio, udev rules, etc.)
